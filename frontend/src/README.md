component = para componentes reusables (navbar,footer, cards)
pages = para paginas completas (Home,Contact)
styles = para los archivos css
assets = para imagenes, iconos ,logos
App = define la estructura principal de la aplicación: Navbar, Footer, y el contenido del “cuerpo” de tu app (como IndexPage).

req = solicitud que envia el cliente al servidor.
res = respuesta del servidor al cliente.
req.body = cuerpo enviado por post/put
req.params = parametros de la url.

fetch = api nativa de react que sirve para realizar peticiones http, basados en promesas

axios= libreria externa para hacer peticiones http , pero con mas funciones.
async = sirve para declara funciones asincronas que devuelven una promesa.
await = se usa dentro de una funcion async para esperar la respuesta de una promesa antes de continuar.
dom = document object model, es una representacion estructurada de una pagina web, (html)
